## вңҰ What is this?

Figma's `.make` files are ZIP archives that bundle design assets, canvas data, and embedded images. This tool **unpacks them and extracts every image at full original resolution** вҖ” no Figma account, no API key, no internet connection required.

> Works with `.make` files exported from Figma's desktop app or collaboration features.

---

## вңҰ Features

| Feature | Details |
|--------|---------|
| рҹ–јпёҸ **Full quality extraction** | Images saved at original resolution, no compression |
| рҹ”Қ **Auto format detection** | PNG, JPEG, GIF, WebP вҖ” detected via magic bytes |
| рҹ“җ **Dimension reading** | Width Г— Height extracted without Pillow |
| рҹ“Ұ **Zero dependencies** | Pure Python stdlib вҖ” `zipfile`, `json`, `struct`, `shutil` |
| рҹ—ӮпёҸ **Metadata parsing** | Reads `meta.json` for file name and export timestamp |
| рҹ–ҘпёҸ **Dual mode** | Run as a script locally **or** as a Colab notebook |
| в¬ҮпёҸ **Colab download** | One-click ZIP download of all images from the notebook |

## вңҰ Supported Formats

| Format | Detection |
|--------|-----------|
| PNG | `\x89PNG\r\n\x1a\n` magic bytes |
| JPEG | `\xff\xd8\xff` magic bytes |
| GIF | `GIF87a` / `GIF89a` |
| WebP | `RIFF` header |
